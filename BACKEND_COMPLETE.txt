🔌 API ENDPOINTS (7 TOTAL):

POST /api/fitness-assessment
   └─ Stores fitness assessment form data
   └─ Called from: index.html (homepage)
   └─ Fields: name, weight, height, goal, fitness_level, email

POST /api/contact
   └─ Stores contact form submissions
   └─ Called from: Contact.html
   └─ Fields: name, email, phone, interest, message

POST /api/bmi
   └─ Stores BMI calculations
   └─ Called from: diet.html
   └─ Fields: weight, height
   └─ Returns: bmi value, category, message

GET /admin/dashboard
   └─ Returns statistics for admin dashboard
   └─ Returns: counts of all submissions

GET /admin/assessments
   └─ Returns all fitness assessments
   └─ Used by: Admin dashboard tab 1

GET /admin/contacts
   └─ Returns all contact submissions
   └─ Used by: Admin dashboard tab 2

GET /admin/bmi-data
   └─ Returns all BMI calculations
   └─ Used by: Admin dashboard tab 3

🚨 TROUBLESHOOTING:

Problem: "Connection refused"
Solution: 
   1. Check if server is running: python app.py
   2. Check if port 5000 is free
   3. Check firewall settings
   4. Try different port in app.py

Problem: "ModuleNotFoundError: No module named 'flask'"
Solution:
   python -m pip install -r requirements.txt

Problem: "Database is locked"
Solution:
   1. Close all connections
   2. Delete fitness_data.db
   3. Restart server (will recreate)

Problem: "CORS error in browser"
Solution:
   1. Check browser console (F12)
   2. Verify API URL matches server
   3. Make sure server is running
   4. Clear browser cache (Ctrl+Shift+Delete)

Problem: Forms not submitting
Solution:
   1. Open browser console (F12)
   2. Check for errors
   3. Verify API_URL in HTML files
   4. Check server is running
   5. Check network tab for failed requests
