# 📘 **1. Project Overview**

The **Fitness & Health Blog** is a visually appealing and user-friendly website containing:

✔ Home Page  
✔ Diet Plans Page  
✔ Exercise Gallery with Modals  
✔ Contact Form Page  

It uses modern UI effects such as:
- Glassmorphism  
- Gradients  
- Smooth hover transitions  
- Responsive layout  

---

# ⭐ **2. Features**

### 🌐 Multi-Page Navigation
Uniform navbar linking to:
- Home  
- Diet Plans  
- Exercise  
- Contact  

### 🥗 Diet Plans
A calorie-based meal table and nutrition tips.

### 💪 Exercise Gallery
Interactive cards:
- Push-ups  
- Squats  
- Plank  
- Yoga  

Each has a **modal popup** showing:
- How to perform  
- Mistakes  
- Variations  
- Pro tips  

### 📩 Contact Page
A beautifully designed contact form for:
- Personal training requests  
- Fitness queries  
- Goal planning  

### 🎨 Modern UI/UX
- Box shadows  
- Animated cards  
- Gradient backgrounds  
- Responsive grids

# 📄 **5. Pages Explained**

## 🏠 **Home Page – index.html**
Contains:
- Welcome message with user introduction  
- Blog purpose  
- Feature cards (Diet, Exercise, Wellness)  
- Stats (50+ plans, 1000+ members, etc.)  
- “Why Choose Us” section  

---

## 🥗 **Diet Plans Page – diet.html**
Features:
- Balanced diet table with calories  
- Glassmorphism cards  
- Nutrition tips (hydration, protein, fiber, etc.)  
- Fully responsive design  

---

## 💪 **Exercise Page – exercise.html**
Includes:
- 4 exercise cards with images  
- Categories (Upper Body, Core, Flexibility, etc.)  
- Detailed modals showing technique, mistakes, variations  

This page uses light JavaScript to open/close modals.

---

## 📩 **Contact Page – contact.html**
Contains:
- Stylish input fields  
- Name, Email, Message fields  
- Submit button animation  
- Back-to-top button  

This page is ready to be connected with backend services like:
- PHP mailer  
- Firebase  
- Node.js  
- Google Forms

# ▶️ **6. How to Run the Project**

### **Option 1 — Open Locally**
Just double-click **index.html**.

### **Option 2 — Use VS Code Live Server**
1. Install **Live Server** extension  
2. Right-click `index.html`  
3. Click **"Open with Live Server"**

# 🧩 **7. How to Modify the Project**

### 🎨 Change Theme Colors  
Edit gradients in the CSS under `body { background: ... }`.

### 🖼 Replace Images  
Update any `<img src="">` paths.

### 🧘 Add More Exercises  
Copy a `.card` block and change:
- Image  
- Name  
- Benefits  
- Modal content  

### 🍎 Add More Diet Plans  
Insert additional rows or new tables in `diet.html`.

### 📩 Connect Form Backend  
Options:
- PHP  
- Node.js  
- FormSubmit  
- Firebase
